Job Title: Software Engineer – Backend (Python)
Company: TechCorp Solutions
Location: Remote / Hybrid

About the Role:
We are looking for a skilled Backend Software Engineer to join our growing team.
You will design and build scalable REST APIs, work with cloud infrastructure,
and collaborate with cross-functional teams to deliver impactful products.

- Experience with PostgreSQL and SQLAlchemy ORM
- Proficiency with Docker and containerization
- Familiarity with Git, GitHub, and version control workflows
- Knowledge of REST API design principles
- Understanding of basic Machine Learning concepts is a plus

Nice to Have:
- Experience with Kubernetes or cloud platforms (AWS/GCP/Azure)
- Familiarity with message queues (Redis, RabbitMQ)
- Knowledge of NLP or LLM integrations
- Open-source contributions
